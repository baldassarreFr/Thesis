# FSSL Foundation Models
This repo host codebase for the PoC project running between Zenseact and ScaleOut on training foundation models for AD perception via Federated Self-Supervised Learning.

## Project Structure

- DINOv1: Code to test DINO on ImageNet
- ZODPretraining: Code necessary to pretrain DINO on ZOD
- evaluation: Code to evaluate trained DINO models on object detection tasks
